# Numery alarmowe i infolinie — Polska — Fact Pack

Pełna lista oficjalnych numerów ratunkowych Polski + infolinie wsparcia psychicznego. **Wszystkie numery są bezpłatne lub w cenie standardowego połączenia.**

## 🚨 Numery alarmowe — służby

| Numer | Czym jest | Operator |
|---|---|---|
| **112** | Numer alarmowy UE (wszystkie służby) | PSP/Policja/Pogotowie |
| **997** | Policja | Komenda Główna Policji |
| **998** | Straż Pożarna | Państwowa Straż Pożarna |
| **999** | Pogotowie ratunkowe | Wojewódzkie stacje pogotowia |
| **991** | Pogotowie energetyczne | Operatorzy sieci dystrybucyjnej |
| **992** | Pogotowie gazowe | PGNiG / PSG |
| **993** | Pogotowie ciepłownicze | Lokalne MPEC |
| **994** | Pogotowie wodno-kanalizacyjne | Lokalne MPWiK |
| **984** | Centrum Powiadamiania Ratunkowego (krzyzysowe) | RCB |
| **987** | WOPR — Wodne Ochotnicze Pogotowie Ratunkowe | Pomoc nad wodą |
| **986** | Straż Miejska | Miejskie/gminne |

> **W razie zagrożenia życia:** dzwoń **112** (działa też z zablokowanej karty SIM).
